# 领域详细内容

本目录包含各领域的 L3 详细内容文件。

## 使用方法

按需生成领域详细内容：
```
/asdm-domain-context-build <领域名>
```

## 可用领域

- agent-domain - Agent域
- workflow-domain - 工作流域
- alert-domain - 告警域
- server-domain - 服务器域
- network-domain - 网络设备域
- knowledge-domain - 知识库域
- remediation-domain - 自动修复域
- infrastructure-domain - 基础设施域

## 文件说明

每个领域目录包含以下文件（按需生成）：
- `index.md` - 领域详情索引
- `entities.md` - 实体定义
- `services.md` - 服务接口
- `apis.md` - API 定义
- `flows.md` - 业务流程
