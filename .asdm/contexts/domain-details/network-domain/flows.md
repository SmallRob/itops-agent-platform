# 网络设备域 - 业务流程

## 1. 设备注册流程

```
用户填写设备信息
       │
       ▼
测试SSH连接 ──── 失败 ──→ 显示错误
       │ 成功
       ▼
测试SNMP连通 ──── 失败 ──→ 显示错误
       │ 成功
       ▼
加密存储密码
       │
       ▼
写入数据库
       │
       ▼
返回设备信息
```

### 关键步骤

1. **连接测试**：SSH + SNMP 双重验证
2. **凭证处理**：从凭证库获取或使用输入的认证信息
3. **密码加密**：AES-256-CBC 加密存储敏感数据
4. **SNMP配置**：自动关联 SNMP 凭证

## 2. SNMP 轮询流程

```
定时器触发(5分钟)
       │
       ▼
查询启用SNMP设备
       │
       ▼
遍历设备列表 ──────────────────┐
       │                      │
       ▼                      │
获取系统信息                    │
       │                      │
       ▼                      │
获取接口列表                    │
       │                      │
       ▼                      │
计算流量速率                    │
       │                      │
       ▼                      │
检测告警条件                    │
       │                      │
       ▼                      │
保存快照+写入历史               │
       │                      │
       ▼                      │
等待200ms ─────────────────────┘
```

### 告警检测规则

- 接口管理 UP 但运行 DOWN
- 入方向错误包激增（>100）

## 3. Trap 处理流程

```
UDP 162端口监听
       │
       ▼
接收Trap报文
       │
       ▼
解析PDU + 提取Varbind
       │
       ▼
保存Trap事件
       │
       ▼
匹配设备(按IP)
       │
       ▼
映射告警级别 ──→ coldStart: critical
       │         warmStart: warning
       ▼         linkDown: warning
写入告警表       linkUp: info
       │         authFailure: high
       ▼
触发告警通知
```

## 4. 设备巡检流程

```
用户触发巡检
       │
       ▼
获取设备信息 + 解密密码
       │
       ▼
创建巡检记录
       │
       ▼
建立SSH连接
       │
       ▼
打开Shell会话
       │
       ▼
关闭分页显示
       │
       ▼
执行巡检命令 ──────────────────┐
       │                      │
       ▼                      │
处理More分页 ──────────────────┘
       │
       ▼
提取命令输出 + 解析结果
       │
       ▼
生成巡检摘要 + 更新历史
       │
       ▼
返回巡检结果
```

### 巡检命令列表（华为设备）

| 命令 | 巡检项 | 解析内容 |
|------|--------|----------|
| display version | 版本信息 | 设备型号、版本号 |
| display interface brief | 接口状态 | 接口列表、状态 |
| display cpu-usage | CPU使用 | CPU利用率 |
| display memory | 内存使用 | 内存利用率 |
| display environment | 环境信息 | 温度、风扇、电源 |

## 6. 告警影响分析流程

```
收到告警
       │
       ▼
获取告警设备
       │
       ▼
查找上游依赖 ──────────────────┐
       │                      │
       ▼                      │
递归遍历 ──────────────────────┘
       │
       ▼
查找下游依赖 ──────────────────┐
       │                      │
       ▼                      │
递归遍历 ──────────────────────┘
       │
       ▼
生成影响报告 + 通知相关人员
```
